## PRD Writing Rules

- PRD ID format: `PRD-###-EnglishTitle`
- Each PRD must generate at least one backlog item.
- Backlog items should carry PRD ID as prefix.
- Completion definition must end with: `체크멍 gate + 가드멍 risk review + 리더멍 final approval`.
