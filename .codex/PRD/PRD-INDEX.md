## PRD 작성 규칙

- PRD ID 형식: `PRD-###-영문제목`
- 모든 PRD는 최소 1개 이상의 실행 항목을 backlog로 전개한다.
- Backlog 항목은 PRD ID를 접두사로 가져야 한다.
- 완료 조건은 "체크멍 게이트 + 가드멍 위험점검 + 리더멍 최종 승인"으로 끝나야 한다.
