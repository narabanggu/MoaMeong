# CODEX 설정

에이전트 실행 루프를 한 페이지로 운영하는 기준 문서다.

## 시작 순서

1. 정책 확인: `.codex/AGENTS.md`
2. 역할/권한 확인: `.codex/agents/team.md`
3. 요청 범위와 산출물 확정

## 표준 실행 루프

1. 현재 요청의 목표와 범위를 확정한다.
2. 최소 단위로 안전하게 변경한다.
3. 결과를 요구 형식으로 보고한다.
4. 블로킹은 인수인계 문서에 즉시 기록한다.

## 상세 운영 문서

- 역할/권한: `.codex/agents/team.md`
- 인수인계: `.codex/agents/handoff.md`
- 실행 절차/에스컬레이션: `.codex/agents/runbook.md`
